# Lab1Web
## 1. Membuat Paragraf
Paragraf merupakan kumpulan dari beberapa kalimat. Paragraf di HTML ditulis dengan tag'<p>. contohnya: 
<html>
  <p>Kami sedang belajar HTML dasar pada matakuliah Pemograman Web.</p>
</html>

untuk merapihkan teks dapat dilakukan seperti ini:
<p align="center"> untuk teks berada di tengah
<p align="justify"> untuk rata kanan kiri
adapun <p align="right"> <p align="left">

## 2. Menambahkan Judul
Heading merupakan sebuah judul yang biasanya digunakan pada sebuah halaman artikel pada web. Untuk menambahkan judul pada HTML dapat diketik dengan <h1> untuk heading 1 sampai tag <h6>. contohnya:
<h1>Heading 1</h1>
<h2>Heading sampai 6</h2>

## 3. Memformatkan Teks
Pada praktikum yang sudah dilakukan, terdapat beberapa tag yang dipakai seperti <b> untuk mempertebal teks, <i> untuk italic text, <span style="color: brown> untuk mewarnai teks.

## 4. Menyisipkan Gambar
untuk menyisipkan gambar juga bisa mnenggunakan link yang berakhiran png, seperti yang saya lakukan pada praktikum ini.
<img src="sertakan link yang memuat gambar", width untuk mengatur lebar gambar

## 5. Menambah Hyperlink
untuk menambah hyberlink dapat diketik seperti di bawah ini, jangan lupa untuk menambah file baru pada visual code agar dapat diakses
<nav>
  <a herf="halaman_1">Halaman Utama</a>
</nav>
